🔒 プライバシーポリシー

最終更新日: 2025年10月26日

重要: TimeDonutは、あなたのプライバシーを最優先に考えています。本アプリはGoogleカレンダーへの読み取り専用アクセスのみを要求し、取得したデータを第三者と共有することは一切ありません。

1. はじめに

TimeDonut（以下「本アプリ」）は、Googleカレンダーの予定を視覚化するmacOSメニューバーアプリケーションです。本プライバシーポリシーは、本アプリがどのような情報を収集し、どのように使用・保護するかについて説明します。

2. 収集する情報

2.1 Googleアカウント情報

本アプリは、Googleカレンダーへのアクセスのために、以下の情報を収集します：

  • ユーザーID: Googleアカウントの一意の識別子
  • メールアドレス: Googleアカウントに関連付けられたメールアドレス
  • アクセストークン: Googleカレンダーへのアクセスに必要な一時的な認証トークン
  • リフレッシュトークン: アクセストークンを更新するための長期トークン

2.2 カレンダーデータ

本アプリは、Googleカレンダーから以下の情報を取得します：

  • 予定のタイトル
  • 開始日時・終了日時
  • 場所（設定されている場合）
  • 説明（設定されている場合）
  • 予定の色・ステータス
重要: 本アプリは読み取り専用のアクセス権限のみを要求します。カレンダーの予定を変更、追加、削除することはできません。

2.3 収集しない情報

本アプリは、以下の情報を収集しません：

  • Googleアカウントのパスワード
  • カレンダー以外のGoogleサービスのデータ
  • 位置情報（予定に記載された場所情報を除く）
  • 連絡先情報
  • デバイスの識別情報（ユーザーIDを除く）
  • 使用状況の分析データ

3. 情報の使用目的

収集した情報は、以下の目的でのみ使用されます：

  • カレンダー表示: Googleカレンダーの予定をドーナツ時計形式で表示
  • 認証: Googleカレンダーへの安全なアクセスを維持
  • 自動同期: 最新の予定情報を定期的に取得
  • 通知: 次の予定までの時間をメニューバーに表示

4. 情報の保存と保護

4.1 クラウドストレージ（Firestore）

以下の情報は、Google Cloud Firestoreに暗号化されて保存されます：

データ 保存場所 暗号化
アクセストークン Google Cloud Firestore ✓ 暗号化済み
リフレッシュトークン Google Cloud Firestore ✓ 暗号化済み
メールアドレス Google Cloud Firestore ✓ 暗号化済み

4.2 ローカルストレージ

以下の情報は、お使いのMac上にローカルで保存されます：

データ 保存場所 暗号化
ユーザーID macOS Keychain ✓ Keychain暗号化
カレンダーデータ（キャッシュ） アプリメモリ ✓ メモリ内暗号化
セキュリティ対策:
  • すべての通信はHTTPS（TLS/SSL）で暗号化されています
  • Firestoreへのアクセスは、認証されたユーザーのみに制限されています
  • アクセストークンは定期的に自動更新され、古いトークンは無効化されます
  • macOS Keychainを使用して、ローカルの認証情報を安全に保存します

5. 情報の共有

本アプリは、収集した個人情報を第三者と共有、販売、貸与することは一切ありません。

ただし、以下の場合には情報を開示する可能性があります：

  • 法的義務: 法令に基づく開示要求がある場合
  • 権利保護: 本アプリまたはユーザーの権利を保護するために必要な場合
  • ユーザーの同意: ユーザーから明示的な同意を得た場合

6. データの保持期間

  • 認証トークン: サインアウトするまで、またはトークンの有効期限（通常7日間）まで保持
  • カレンダーデータ: アプリ実行中のメモリ内にのみ保持され、アプリ終了時に削除
  • ユーザーID: サインアウトするまで、またはアプリをアンインストールするまで保持

7. ユーザーの権利

ユーザーには、以下の権利があります：

7.1 アクセス権

保存されている個人情報を確認する権利があります。

7.2 削除権

本アプリ内の「サインアウト」機能を使用することで、以下のデータが削除されます：

  • ローカルに保存されたユーザーID（Keychain）
  • メモリ内のカレンダーデータ

Firestoreに保存された認証トークンを削除するには、Googleアカウントのアプリの権限管理ページから「TimeDonut」へのアクセスを取り消してください。

7.3 データポータビリティ

カレンダーデータは、元々Googleカレンダーに保存されているため、Google Takeoutを使用してエクスポートできます。

8. 子供のプライバシー

本アプリは、13歳未満の子供を対象としていません。13歳未満の子供から故意に個人情報を収集することはありません。保護者の方で、お子様が本アプリを使用して個人情報を提供したと思われる場合は、速やかにご連絡ください。

9. Google APIの使用

本アプリは、Google Calendar APIを使用してカレンダーデータにアクセスします。Googleのプライバシーポリシーについては、Googleプライバシーポリシーをご確認ください。

本アプリのGoogle APIの使用は、Google API Services User Data Policyに準拠しています。特に、以下の原則を遵守しています：

  • 限定的使用: カレンダーデータは、本アプリの機能提供のみに使用されます
  • 第三者への非開示: カレンダーデータを第三者と共有しません
  • 人間による読み取り禁止: 開発者を含む人間が、ユーザーのカレンダーデータを読み取ることはありません

10. クッキーとトラッキング

本アプリは、クッキーや分析ツールを使用しません。ユーザーの行動を追跡することはありません。

11. プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更やアプリの機能更新に伴い、随時更新されることがあります。重要な変更がある場合は、本ページの「最終更新日」を更新し、アプリ内で通知します。

12. お問い合わせ

本プライバシーポリシーに関するご質問、ご懸念、またはデータ削除のご要望は、以下までご連絡ください：

  • GitHub Issues: https://github.com/tomohiro-owada/TimeDonut/issues
  • メール: お問い合わせフォームをGitHub Issuesよりご利用ください

13. 準拠法と管轄

本プライバシーポリシーは、日本国の法律に準拠し、解釈されます。本ポリシーに関する紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とします。

要約:

  • ✅ Googleカレンダーへの読み取り専用アクセスのみ
  • ✅ データは暗号化されて保存
  • ✅ 第三者とデータを共有することは一切ありません
  • ✅ クッキーや分析ツールは使用しません
  • ✅ いつでもサインアウトしてデータを削除可能
