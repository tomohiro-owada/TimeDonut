🍩 TimeDonut

Googleカレンダーをドーナツ時計で表示するmacOSメニューバーアプリ

✨ TimeDonutとは？

TimeDonutは、Googleカレンダーの予定を美しいドーナツ型の24時間時計で可視化するmacOS専用のメニューバーアプリです。次の予定までの時間を一目で確認でき、時間管理をよりスムーズにします。

🕐 24時間ドーナツ時計

予定を24時間時計のドーナツ型で表示。直感的に一日の予定を把握できます。

⏱️ カウントダウン表示

次の予定までの残り時間をメニューバーにリアルタイム表示。時間を有効活用できます。

📅 Googleカレンダー連携

Googleカレンダーと自動同期。複数のカレンダーの予定を統合して表示します。

🎨 カラフルな予定表示

Googleカレンダーの色分けをそのまま反映。予定の種類が一目瞭然です。

📜 マーキー表示

長い予定名もメニューバーでスクロール表示。すべての情報が確認できます。

🔐 セキュアな認証

Google Cloud Functionsを使用した安全なOAuth認証。トークンはFirestoreに暗号化保存されます。

💻 動作環境

  • macOS 12.0 (Monterey) 以降
  • Googleアカウント（Googleカレンダーへのアクセス権限が必要）
  • インターネット接続

🚀 使い方

  1. アプリを起動 - TimeDonutを起動すると、メニューバーに時計アイコンが表示されます
  2. Googleアカウントでサインイン - 初回起動時にGoogleアカウントでの認証が必要です
  3. カレンダーアクセスを許可 - Googleカレンダーへの読み取り専用アクセスを許可します
  4. 予定を確認 - メニューバーアイコンをクリックすると、ドーナツ時計で一日の予定が表示されます
  5. 自動更新 - 5分ごとに自動的にカレンダーと同期します

🔧 主な機能

  • メニューバーに次の予定までの時間と予定名を表示
  • 24時間ドーナツ型時計で予定を可視化
  • 開催中の予定は「開催中」と表示
  • 予定の詳細（タイトル、時刻、場所、説明）をポップオーバーで確認
  • 右クリックメニューから更新・サインアウト・終了が可能
  • リフレッシュボタンで手動同期

🔒 プライバシーとセキュリティ

TimeDonutはあなたのプライバシーを重視しています：

  • Googleカレンダーへの読み取り専用アクセスのみ要求
  • トークンはGoogle Cloud Firestoreに暗号化して保存
  • ユーザーIDのみをローカルのKeychainに保存
  • カレンダーデータはローカルにキャッシュされますが、暗号化されています
  • 第三者とデータを共有することはありません
